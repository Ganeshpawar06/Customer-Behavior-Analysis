## 🛠️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Ganeshpawar06/Customer-Behavior-Analysis.git
cd Customer-Behavior-Analysis
```

Or visit the repository directly:

🔗 https://github.com/Ganeshpawar06/Customer-Behavior-Analysis :contentReference[oaicite:0]{index=0}

---

### 2. Open the Jupyter Notebook

Launch the **Customer_Shopping_Behavior_Analysis.ipynb** notebook.

The notebook includes the following stages:

- Import the customer shopping dataset
- Perform exploratory data analysis (EDA)
- Clean and preprocess the data
- Establish a connection with the SQL database
- Export the processed dataset to SQL

---

### 3. Import Data into SQL

Create a database using your preferred SQL platform:

- MySQL
- PostgreSQL
- Microsoft SQL Server

Run the Python notebook to transfer the cleaned dataset into the SQL database.

---

### 4. Execute SQL Analysis

Open the **customer_behavior_sql_queries.sql** file.

Use the provided SQL scripts to answer important business questions, including:

- Customer segmentation
- Shopping behavior
- Purchase frequency
- Spending analysis
- Loyalty trends
- Product performance
- Customer demographics

---

### 5. Build the Power BI Dashboard

Connect Power BI to your SQL database.

Open the **customer_behavior_dashboard.pbix** file and create an interactive dashboard featuring:

- Sales KPIs
- Customer insights
- Demographic analysis
- Product performance
- Purchase trends
- Interactive slicers and filters

---

### 6. Prepare Business Deliverables

Complete the project by creating:

- 📄 A detailed business analytics report summarizing findings
- 📊 A professional presentation highlighting insights and recommendations
- 🎨 A visually engaging presentation deck using Gamma AI (optional)
